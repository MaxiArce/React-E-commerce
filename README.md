## About

Final project of the ReactJs course for [CoderHouse](https://www.coderhouse.es/).

## Demo

[Live Demo](https://react-ecommerce-80cbd.web.app/)

## Features

* Register / Login.
* View products by category.
* Add products to cart.
* Shop as a guest.
* Add Favorites.

## Built using

* React
* Taildwind CSS
* React Router Dom

## Build/Run

* npm install
* npm start run
